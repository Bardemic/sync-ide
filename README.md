# sync-ide

One CLI command to open any interface for coding from your active path.

Agent Orchestrators:
- T3 Code
- Codex
- Cursor-Glass
- OpenCode GUI
- Superset

IDEs:
- Cursor
- Orchids
- Virtual Studio Code
- Zed

Else:
- Github Desktop

## Install

```bash
npm install -g ide-open
```

## Usage

```bash
cd ~/repos/my-project
ide [choice]      # cursor, t3, zed, ghd, codex, cursor-glass, orchids, vscode, opencode, superset
ide list          # show every supported provider
ide --help
```
